# Первый проект в Frontend.

Это первый проект на HTML и CSS. В папке два отдельных HTML-документа с соответствующими CSS-стилями.

## Содержание

- `index.html` — страница, где я практиковался в построении сайта с использованием в основном - Flexbox.
- `style.css` — стили для `index.html`.
- `indexd.html` — страница, где я использовал CSS Grid для размещения элементов на экране.
- `style2.css` — стили для `indexd.html`.

## Цель проекта

- Научиться базовой разметке HTML.
- Попрактиковаться с Flexbox и Grid для построения макета страниц.
